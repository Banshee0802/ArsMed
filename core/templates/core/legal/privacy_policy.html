Политика конфиденциальности

Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки персональных данных пользователей сайта ArsMedica (далее — «Сайт»), созданного в рамках учебного проекта.

1. Общие положения

1.1. Проект ArsMedica является учебным и не осуществляет коммерческую деятельность.

1.2. Пользователь предоставляет данные добровольно при использовании функционала сайта (если таковой предусмотрен).

2. Собираемые данные

2.1. Сайт может собирать минимальные технические данные, такие как IP-адрес, cookies и технические параметры браузера — только для корректной работы сайта.

3. Использование данных

3.1. Все данные используются исключительно для обеспечения работы сайта, улучшения его технических характеристик и анализа посещаемости.

4. Защита данных

4.1. Разработчик предпринимает разумные меры для защиты данных от несанкционированного доступа, однако не гарантирует абсолютную безопасность, учитывая учебный характер проекта.

5. Передача данных третьим лицам

5.1. Данные не передаются третьим лицам, за исключением случаев, предусмотренных законодательством.

6. Заключительные положения

6.1. Разработчик может обновлять Политику без уведомления Пользователя.
